## ⚙️ Como executar o projeto

Clone esse repositório:

```bash
git clone https://github.com/4L1C3-R4BB1T/todo-list.git
```

Entre na pasta backend do projeto:

```bash
cd ./backend
```

Execute o container docker:

```bash
docker compose up -d
```

Execute o projeto:

```bash
./mvnw spring-boot:run
```
