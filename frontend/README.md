## ⚙️ Como executar o projeto

Clone esse repositório:

```bash
git clone https://github.com/4L1C3-R4BB1T/todo-list.git
```

Entre na pasta frontend do projeto:

```bash
cd ./frontend
```

Instale as dependências do projeto:

```bash
npm install
```

Execute o projeto:

```bash
npm start
```
